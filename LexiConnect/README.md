# LexiConnect – Location-Based Lawyer Discovery & Appointment Platform

This is a university group project for a Sri Lankan legal appointment and compliance platform.

## Project Structure

- `backend/`  → FastAPI + PostgreSQL (Python)
- `frontend/` → React (Vite) + Tailwind CSS + Axios
- `docs/`     → Reports, task breakdown, supervisor notes
- `diagrams/` → UI wireframes, ERD, flowcharts

## Interim Focus (by Dec 11)

- User Registration & Login (JWT)
- Search lawyers (district, city, specialization, language)
- View lawyer profile
- Book appointment
- Manage bookings (client)
